# Collaborative Work

During this step, you will produce a LaTeX document together with the other participants in the training course.
The output will take the form of **a GitHub repository** in the `SCRIPT-SIE-2026` working group.
We provide LaTeX file templates in the following repository: [here](https://github.com/SCRIPT-SIE-2026/SCRIPT_SIE_2026_05_04_Git_Templates.git).

## Prerequisites

With your GitHub account, you must be a member of the `SCRIPT-SIE-2026` organization.

## Demonstration

- Create a repository on GitHub.
- Add members.
- Clone your repository to your local machine.

## Constraints

Here are the different constraints we will impose:

- Groups must be composed of at least 3 people.
- Each group will work on a shared repository, and the group's final output must be on the `main` branch.
- In addition to the `main` branch, there must be one branch per person.
- Each person's contributions must be integrated through `pull requests`.
- A `README.md` file must be present to introduce the members and their roles in the project.
