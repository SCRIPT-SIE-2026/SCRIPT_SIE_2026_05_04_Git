# Travail collaboratif

*An english version of this document is available [here](PHASE_3_eng.md).*

Durant cette étape, vous devrez produire un document LaTeX en commun avec les autres personnes de la formation.
La production prendra la forme **d'un dépôt sur GitHub** dans le groupe de travail `SCRIPT-SIE-2026`.
Nous vous fournissons des modèles de fichiers LaTeX disponibles dans le dépôt suivant : [ici](https://github.com/SCRIPT-SIE-2026/SCRIPT_SIE_2026_05_04_Git_Templates.git).

## Prérequis

Avec votre compte GitHub, vous devez être membre de l'organisation `SCRIPT-SIE-2026`.

## Démonstration

- Créer un dépôt sur GitHub.
- Ajouter des membres.
- Cloner son dépôt sur sa machine locale.

## Contraintes

Voici les différentes contraintes qu'on s'impose :

- Les groupes doivent être composés d'au moins 3 personnes.
- Chaque groupe travaillera sur un dépôt partagé et on retrouvera le rendu final du groupe sur la branche `main`.
- En plus de la branche `main`, on doit retrouver une branche par personne.
- L'intégration des contributions de chacun doit être faite sous la forme de `pull requests`.
- Un fichier `README.md` doit être présent pour présenter les membres et leurs rôles dans le projet.
