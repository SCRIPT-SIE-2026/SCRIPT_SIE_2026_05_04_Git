# Travail collaboratif

Durant cette étape, vous devrez produire un document Latex en commun avec les autres personnes de la formation. 
La production prendra la forme **d'un dépôt sur Github** dans le groupe de travail `SCRIPT-SIE-2026`.
Nous vous fournissons des templates de fichiers Latex disponibles sur le dépôt suivant [ici](https://github.com/SCRIPT-SIE-2026/SCRIPT_SIE_2026_05_04_Git_Templates.git).

## Prérequis

Vous devez avec votre compte Github être membre de l'organisation `SCRIPT-SIE-2026`


## Démonstration

- Créer d'un dépôt sur Github
- Ajouter des membres
- Clone son dépôt sur sa machine locale


## Contraintes 

Voici les différentes contraintes qu'on s'impose :

- Les groupes doivent être au moins composés de 3 personnes.
- Chaque groupe travaillera sur un dépôt partagé et on retrouvera le rendu final du groupe sur la branche `main`. 
- En plus de la branche `main`, on doit retrouver une branche par personne.
- L'intégration des contributions de chacun doit être faite sous la forme de `pull-request`.
- Un fichier README.md doit être présent pour présenter les membres et leurs rôles dans le projet.
